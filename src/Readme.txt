Day-32-React-Shopping-Card-Redux

This project is a simple e-commerce product listing application built with React, Redux Toolkit, and Material-UI. 
It allows users to view a list of products, add products to the cart, and manage the quantity of items in the cart.

## Features

- **Product Listing**: Displays a list of products in a card layout.
- **Add to Cart**: Users can add products to their cart.
- **Manage Cart**: Users can increase or decrease the quantity of products in their cart.
- **Responsive Design**: The UI is responsive and adapts to different screen sizes.

## Tech Stack

- **React**: For building the user interface.
- **Redux Toolkit**: For managing the application's state.
- **Material-UI (MUI)**: For styling and building UI components.
- **Vite**: For fast development and bundling.
